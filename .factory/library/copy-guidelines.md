# Copy Guidelines

Shared wording and visual-language rules for this mission.

**What belongs here:** UI wording, label consistency, icon-usage rules.
**What does NOT belong here:** visual specs from other repos or one-off marketing copy.

---

- Use simple, direct wording.
- Keep wording consistent across portals for the same entity or action.
- Do not use emojis.
- Use fewer icons; only keep an icon when it materially improves comprehension.
- Avoid decorative or novelty labels.
- Prefer action labels that match backend/business meaning exactly.
